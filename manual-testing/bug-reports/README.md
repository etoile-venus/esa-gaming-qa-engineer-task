# Bug Reports

This folder is for bug reports found during testing.

Use Markdown files here to describe the issue, steps to reproduce, expected result, actual result, severity, priority, status, test environment, and attachments.

When a bug is found during an executed test case, reference the related test case instead of repeating the same preconditions. Add a separate Preconditions section only when there is no related test case, or when the bug needs setup that is not covered by the related test case.
