# Test Cases

This folder is for manual test cases grouped by application feature.

Use Markdown files here to write test cases before converting them to PDF or XLSX if needed.

## Structure

- `auth/registration.md` - account registration checks.
- `auth/login.md` - login, logout, and account access checks.
- `catalog/search-and-filter.md` - search, filters, sorting, and product result checks.
- `buying/cart.md` - add to cart, cart updates, removals, and totals.
- `buying/checkout.md` - checkout steps, delivery details, and validation.
- `buying/payment.md` - payment flow, payment validation, and order confirmation.

Test cases will be added after each flow is reviewed.
